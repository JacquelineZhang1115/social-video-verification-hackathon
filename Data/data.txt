Extract the zip file here (the ./data/ directory)

It contains at set of .mat files. 
Expected naming is: mouth-data-fake<N>-ID<PERSON>.mat,
where N is the fake camera number (code expects 2,3, or 4), 
and PERSON is the participant number (code expects 1 thru 25, except 17). 

Eg: mouth-data-fake3-ID11.mat

These data contain facial tracking data for all participants. 
Each face is tracked with a set of ~70 landmarks per video frame.
